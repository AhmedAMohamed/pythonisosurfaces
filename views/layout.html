Info
Iso-surfaces are generated in real time serverside via a pure Python program that implements the marching cubes algortihms. It can read VTK files. It uses processing.js client side. The processing code is generated dynamically serverside and executed clientside. It can hadle about 2000 primitives (lines, triangles, spheres) per image.
